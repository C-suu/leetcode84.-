### （1）题目意思与生僻概念解释

**题目意思：**
在一排宽度固定为 1、高度各异的柱子中，找出能够被包含在内的、面积最大的实心矩形。

**核心生僻概念解释：**

  * **单调栈的基础应用（找最近较小值）：** 单调栈最经典的功能，就是用来寻找数组中某个元素**左边或右边第一个比它小（或大）的元素**。
  * **边界数组（`left` 和 `right`）：**
      * `left` 数组：专门用来记录每根柱子**左侧**最近的一根**比它矮**的柱子的索引位置。
      * `right` 数组：专门用来记录每根柱子**右侧**最近的一根**比它矮**的柱子的索引位置。

-----

### （2）代码解题思路解析

这段代码摒弃了烧脑的哨兵技巧，采用了**拆解思维（两趟遍历 + 一次结算）**，逻辑极其清晰：

1.  **明确核心目标：** 要想找到最大矩形，就必须算出“以每一根柱子作为绝对高度（瓶颈）”时，矩形能向左右延伸多远。
2.  **第一步（求左边界）：** 从左向右遍历每一根柱子。利用单调递增栈，把左边比当前柱子高（或相等）的柱子全部踢掉，栈顶剩下的就是左边第一个比当前矮的柱子。将其索引记入 `left` 数组。如果栈空了，说明左边没有比它矮的，默认边界为 `-1`（虚拟的最左侧边界）。
3.  **第二步（求右边界）：** 清空栈，从右向左倒序遍历每一根柱子。采用同样的单调递增栈逻辑，把右边较高（或相等）的柱子踢掉，栈顶剩下的就是右边第一个比当前矮的柱子。将其索引记入 `right` 数组。如果栈空了，说明右边一直到头都没有比它矮的，默认边界为 `n`（即数组长度，虚拟的最右侧边界）。
4.  **第三步（计算面积）：** 此时，所有柱子的左边界和右边界都已明了。只需遍历一次，套用公式 `宽度 = 右边界 - 左边界 - 1`，再乘以当前柱子的高度，即可求出最大面积。

### （3）带有注释的代码框

```python
class Solution:
    def largestRectangleArea(self, heights: List[int]) -> int:
        n = len(heights)
        # 初始化 left 数组，默认左边界全部为 -1
        left = [-1] * n  
        st = []
        # 第一趟：从左到右遍历，寻找每根柱子左侧第一个比它矮的柱子
        for i, h in enumerate(heights):
            # 当栈不为空，且栈顶元素高度 >= 当前高度时，弹出栈顶
            while st and heights[st[-1]] >= h:
                st.pop()
            # 如果栈未空，说明找到了左侧较矮的柱子，记录其索引
            if st:
                left[i] = st[-1]
            # 当前柱子索引入栈
            st.append(i)
        # 初始化 right 数组，默认右边界全部为 n（数组总长度）
        right = [n] * n  
        # 清空栈，准备第二趟遍历
        st.clear()
        # 第二趟：从右到左遍历，寻找每根柱子右侧第一个比它矮的柱子
        for i in range(n - 1, -1, -1):
            h = heights[i]
            # 同样保持栈的单调递增性
            while st and heights[st[-1]] >= h:
                st.pop()
            # 记录右侧较矮柱子的索引
            if st:
                right[i] = st[-1]
            st.append(i)
        ans = 0
        # 第三趟：同时遍历高度、左边界、右边界，计算最大面积
        for h, l, r in zip(heights, left, right):
            # 面积 = 高度 * (右边界 - 左边界 - 1)
            ans = max(ans, h * (r - l - 1))
        return ans
```

### （4）每一行代码详细中文解释

1.  `class Solution:`
      * **解释：** 定义名为 `Solution` 的解题类。
2.  `def largestRectangleArea(self, heights: List[int]) -> int:`
      * **解释：** 定义核心方法，接收高度数组 `heights`，返回最大面积整数。
3.  `n = len(heights)`
      * **解释：** 获取柱子的总数量，赋值给变量 `n`。
4.  `left = [-1] * n`
      * **解释：** 创建一个长度为 `n` 的列表 `left`，全部填充 `-1`。`-1` 代表在数组最左边有一道无形的、极矮的墙。
5.  `st = []`
      * **解释：** 初始化一个空列表，用作单调递增栈。
6.  `for i, h in enumerate(heights):`
      * **解释：** 从左往右正序遍历所有柱子。`i` 是当前索引，`h` 是当前柱子的高度。
7.  `while st and heights[st[-1]] >= h:`
      * **解释：** 只要栈里还有元素，并且栈顶索引代表的柱子高度大于或等于当前高度 `h`，就开启循环。
8.  `st.pop()`
      * **解释：** 将不满足单调递增（被挡住视线）的栈顶元素无情弹出。
9.  `if st:`
      * **解释：** 经过上面的弹出操作后，如果栈里还有剩下元素。
10. `left[i] = st[-1]`
      * **解释：** 栈顶剩下的这个索引，就是当前柱子左侧第一个比它矮的柱子，将其记录到 `left` 数组中。
11. `st.append(i)`
      * **解释：** 无论如何，将当前柱子的索引 `i` 压入栈中，供后续柱子参考。
12. `right = [n] * n`
      * **解释：** 创建一个长度为 `n` 的列表 `right`，全部填充 `n`。`n` 刚好超出数组最大索引，代表右侧无穷远处极矮的墙。
13. `st.clear()`
      * **解释：** 将单调栈清空，为接下来的反向遍历做准备。
14. `for i in range(n - 1, -1, -1):`
      * **解释：** 使用 `range` 函数生成从 `n-1` 到 `0` 的倒序索引序列，开始从右往左倒序遍历。
15. `h = heights[i]`
      * **解释：** 获取倒序遍历时当前索引 `i` 对应的高度 `h`。
16. `while st and heights[st[-1]] >= h:`
      * **解释：** 同理，维护单调递增栈，将阻挡视线的高柱子弹出。
17. `st.pop()`
      * **解释：** 执行弹出操作。
18. `if st:`
      * **解释：** 若弹出后栈不为空。
19. `right[i] = st[-1]`
      * **解释：** 记录栈顶索引，即当前柱子右侧第一个比它矮的柱子索引，存入 `right` 数组。
20. `st.append(i)`
      * **解释：** 将当前柱子索引入栈。
21. `ans = 0`
      * **解释：** 初始化历史最大面积记录变量 `ans` 为 0。
22. `for h, l, r in zip(heights, left, right):`
      * **解释：** 利用 `zip` 函数，将高度数组、左边界数组、右边界数组打包。每次循环同时取出对应同一根柱子的：高度 `h`、左边界 `l`、右边界 `r`。
23. `ans = max(ans, h * (r - l - 1))`
      * **解释：** 计算这根柱子能撑起的最大面积（高度乘以左右边界的间距），如果新面积大于 `ans`，则覆盖更新。
24. `return ans`
      * **解释：** 返回最终记录到的最大矩形面积。
